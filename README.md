## Hoe werkt deze training?

1. **Fork** deze repository naar je eigen GitHub account
2. **Clone** je fork naar je computer
3. Volg de lessen in de `lessen/` map
4. Doe de oefeningen in de `oefeningen/` map
5. Gebruik de [cheatsheet](cheatsheet.md) als geheugensteun

## Wat heb je nodig?

- Een **GitHub account** (gratis) - [github.com/signup](https://github.com/signup)
- **Git** geinstalleerd op je computer - [git-scm.com](https://git-scm.com/)
- Een **code editor** (bv. VS Code) - [code.visualstudio.com](https://code.visualstudio.com/)
